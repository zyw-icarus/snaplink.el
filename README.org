* snaplink.el

[[https://github.com/doomemacs/core][https://img.shields.io/badge/doom-emacs-blue.svg?style=for-the-badge]]

=snaplink.el= is a Emacs plugin to upload a local image path at point with =rclone= and
replace that path with a public URL in place.

Supported link forms:

- Org:  =[[./assets/a.png][]]=, =[[file:./assets/a.png][]]=
- Markdown image: =![](./assets/a.png)=
- Markdown link: =[](./assets/a.png)=

Path rules:

- Paths are interpreted relative to the current file.
- =foo/bar.png= and =./foo/bar.png= are supported.
- Any path containing =../= is rejected.
- Absolute paths are rejected.

** Configuration

#+begin_src elisp
;; Install once:
;; (package-vc-install "https://github.com/zyw-icarus/snaplink.el")

(require 'snaplink)

(setq snaplink-rclone-remote "r2:your-bucket"
      snaplink-public-base-url "https://img.example.com")
#+end_src

Doom Emacs example:

#+begin_src elisp
;; ~/.doom.d/packages.el
(package! snaplink
  :recipe (:host github :repo "zyw-icarus/snaplink.el"))
#+end_src

#+begin_src elisp
;; ~/.doom.d/config.el
(use-package! snaplink
  :commands (snaplink-upload-at-point)
  :init
  (map! :leader
        :desc "Snap local image link"
        "n u" #'snaplink-upload-at-point)
  :config
  (setq snaplink-rclone-remote "r2:your-bucket"
        snaplink-public-base-url "https://img.example.com"))
#+end_src

Then run =doom sync= and restart Emacs.
